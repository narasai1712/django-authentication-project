# Django Authentication System 

A simple and professional Django Authentication project with Signup, Login, and Dashboard functionality, built using Django, HTML and CSS.

This project demonstrates user authentication, session handling, protected routes, and clean UI design.


##  Features
- User Signup
- User Login
- Authentication using Django auth system
- Protected Dashboard (login required)
- Logout functionality
- Error handling (invalid credentials, user exists)
- Clean and modern UI


##  Tech Stack
- Backend: Python, Django
- Frontend: HTML, CSS
- Database: SQLite
- Tools: Git, GitHub
